🔧 Специфичные тактики по движкам

Google

  • Core Web Vitals — LCP < 2.5 с, CLS < 0.1
  • E-E-A-T — авторские био, даты, цитаты
  • Избранные сниппеты — FAQ для «что такое / как / стоимость»
  • Внутренние ссылки — с топовых страниц на 11-20
  • A/B тесты заголовков — высокие показы, низкий CTR
  • Мобильная адаптивность — пройти Mobile-Friendly Test

Яндекс

  • Турбо-страницы — включить для всех в зоне 6-20
  • Поведенческие факторы — время на сайте, отказы
  • Гео-таргетинг — регионы в Вебмастере, Яндекс.Карты
  • Синергия с Директом — параллельные кампании для 6-10
  • Свежесть контента — обновлять статьи ежеквартально
  • Schema.org — Organization, BreadcrumbList, FAQPage
